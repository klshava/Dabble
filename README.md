# Dabble
Commercial and Customer Overview PBI Dashboard

A Power BI portfolio project built to demonstrate BI analyst capabilities relevant to [Dabble](https://www.dabble.com.au) — Australia's social-first sports betting platform.

## Project Overview

This project simulates the kind of analytics a BI analyst at Dabble would produce day-to-day. It covers commercial performance, customer segmentation, and time-based intelligence, all presented in a two page Power BI report styled with Dabble's brand colours.
